Detailed Description

Defines basic sized integer types. On C++11, we use "stdints". On other platforms, we define our own.

Namespace for TDOG implementation machinery.

Enumeration Type Documentation

enum tdog::report_style_t

An enumerated type used to designate the style of test run reports.

Non-verbose reports contain only summary information. Verbose reports, on the other hand, contain maximum trace information. For verbose XML, the report will contain test case output information within the "<system-out>" element. HTML reports are suitable for viewing in a browser.
